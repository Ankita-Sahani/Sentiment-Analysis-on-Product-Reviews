# Sentiment-Analysis-on-Product-Reviews
Sentiment Analysis on Amazon Product Reviews using Python, TextBlob, NLP techniques, and data visualization to classify customer feedback into Positive, Negative, and Neutral sentiments.

# Dataset
Link - https://www.kaggle.com/datasets/snap/amazon-fine-food-reviews
The Amazon Fine Food Reviews dataset is a comprehensive collection of customer feedback that enables effective sentiment analysis. Its large size, detailed review content, and rating information make it an excellent dataset for studying customer opinions and building NLP-based sentiment classification systems.
